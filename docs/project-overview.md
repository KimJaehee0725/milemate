# 프로젝트 개요

## 주제
기획자를 위한 기술 개발 난이도 조사 어시스턴트 에이전트

## 문제의식
기술 이해도가 낮은 기획자는 아이디어를 실제 개발 과제로 전환하는 과정에서 다음 어려움을 겪는다.
- 어떤 기술 요소가 필요한지 모름
- 무엇이 기술적으로 어려운지 판단하기 어려움
- 핵심 난제를 정의하기 어려움
- 대체 구현 경로와 MVP 우회 전략을 떠올리기 어려움

## 에이전트 역할
사용자가 아이디어를 입력하면 다음을 지원한다.
1. 아이디어를 구체적 문제 정의로 구조화
2. 필요한 기술 요소를 분해
3. 개발 난이도를 평가
4. 핵심 기술 난제를 정의
5. 우회 전략 및 MVP 경로를 제안
6. 추가 조사 키워드와 자료 방향을 안내

## 현재 상태
- 문제 정의 초안 수립
- 도메인 선택 논의 필요
- 평가 기준 및 시스템 설계 미정
